# Change: create-subjects-table

## Why
Criar a base de dados para subjects, permitindo que cada usuário registre e gerencie suas disciplinas acadêmicas com propriedade definida, viabilizando controles de acesso e futuras automações.

## What Changes
Adicionar nova tabela subjects com campos para nome, código, descrição, status e referência ao usuário.

## Impact
Permite gestão de disciplinas acadêmicas, melhorando a organização dos usuários no EduTrack AI.
