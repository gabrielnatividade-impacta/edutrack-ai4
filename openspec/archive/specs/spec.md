# feature-notas-atividades Specification

## Purpose
Define the functionality to allow teachers to assign grades to students for specific activities in EduTrack AI.

## ADDED Requirements
### Requirement: Allow teachers to submit grades for activities
The system SHALL allow authenticated teachers to submit grades for students in specific activities.
#### Scenario: Teacher submits a grade
- **WHEN** teacher selects an activity and student
- **THEN** system records the grade with validation
