## Why

Students need a way to track their academic progress by calculating completion percentages for their tasks. This will help them understand how much work they've completed and how much remains, providing motivation and better time management.

## What Changes

- Create a new Python script `scripts/calculate_progress.py` that:
  - Calculates progress percentage (completed tasks / total tasks)
  - Returns the result as JSON format
  - Can be executed from command line or imported as a module

## Capabilities

- `progress-calculation`: Enable calculation of academic task completion progress and return results in JSON format
