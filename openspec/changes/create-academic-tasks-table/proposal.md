# Change: create-academic-tasks-table

## Why
Criar a tabela `academic_tasks` para permitir que alunos registrem e gerenciem obrigações acadêmicas (lições, provas, trabalhos) vinculadas a disciplinas.

## What Changes
Adicionar nova tabela `academic_tasks` com campos para título, descrição, data de vencimento, status e referência à disciplina.

## Impact
Permite gestão de tarefas acadêmicas, melhorando a organização dos alunos no EduTrack AI.
